Used to determine which cardinal direction the player is trying to move in.

Given that the system cannot know exactly which direction the player is moving in when they select a destination, it must mathematically calculate what the player is trying to do. This function will analyze the requested destination in relation to the Character's current position. It will then return the direction the player is trying to move in, or 'Invalid' if this is not one of the allowed directions.

First, checks if the location is within the bounds of the map.

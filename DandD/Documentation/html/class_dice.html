§ roll()

int Dice::roll ( std::string  exp)

Will 'roll' a die using the string expression that is passed to it. String must be in the format "xDy" with option modifier at the end "xDy[+z]". x and y MUST be positive integers. z can be positive or negative, as per modifiers in the d20 rules. y must be one of [4, 6, 8, 10, 12, 20, 100].

Returns -1 if invalid string is provided.

Parameters
stringexp, the string representing the dice roll to perform
Returns
integer, a random number conforming to the requested dice roll

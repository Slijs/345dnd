Detailed Description

This class is a singleton class designed to handle the text input output for the engine this class also complements the singleton class SingletonFilePathAndFolderManager in managing input output to the correct files for campaign editing, which involves creating, editing and deleting campaigns and maps within campaigns, also loading some prebuilt deafult maps if the user wants to.
